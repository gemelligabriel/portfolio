# Welcome to my Portfolio!

This project was developed to showcase my projects, skills, and experience as a developer using modern front-end technologies.

## Technologies Used

- React
- TypeScript
- Vite
- CSS / Tailwind
- ESLint

## About the Project

The goal of this portfolio is to gather my main projects, experiences, and skills in one place, providing a modern, responsive, and intuitive navigation experience.

In addition, this project represents my growth as a developer, exploring concepts such as:

- Component-based architecture
- Responsiveness
- UI/UX best practices
- Code organization
- Performance with Vite
- Type safety with TypeScript

## How to Run the Project

```bash
# Navigate to the project folder
cd portfolio

# Install dependencies
npm install

# Run the project
npm run dev
````

The project will be available at:
[http://localhost:5173](http://localhost:5173)
